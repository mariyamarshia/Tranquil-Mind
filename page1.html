Tetris

Tetris is a tile-matching puzzle game, developed in the Soviet Union in 1984. The game has a simple goal of destroying lines of blocks before it reaches the top. The line is made up of a square block. The Tetrimino is the shape of the 4 connected blocks that fall down.

Start Playing!

Snake Game

In the game of Snake, the player uses the arrow keys to move a "snake" around the board. As the snake finds food, it eats the food, and thereby grows larger. The game ends when the snake either moves off the screen or moves into itself. The goal is to make the snake as large as possible before that happens

Start Playing!
